# VectorShift Frontend Technical Assessment

This repository contains the solution for the frontend technical assessment.

## Project Structure

- `frontend/`: React application (React Flow, Zustand)
- `backend/`: FastAPI application

## How to Run

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm start
```
